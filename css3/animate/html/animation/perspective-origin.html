perspective-origin

语法

perspective-origin: [ [ <percentage> | <length> | left | center | right ] [ <percentage> | <length> | top | center | bottom ]? ] | [ [ left | center | right ] || [ top | center | bottom ] ] ;
语法项目 说明
初始值 50% 50%
适用于 块元素和行内元素
可否继承 否
取值 长度值
媒介 视觉
版本 CSS3.0

说明

perspective-origin属性定义在X轴和Y轴的3D元素。这个属性允许你改变3D元素的底部位置。定义时的perspective-origin属性，它是一个元素的子元素，透视图，而不是元素本身。
提示：使用此属性必须和perspective属性一起使用，只影响3D转换的元素！

受影响变形函数：perspective()。

取值

该属性提供2个参数值，第一个用于横坐标，第二个用于纵坐标；如果只提供一个，该值将用于横坐标，纵坐标将默认为50%。

percentage：用百分比指定坐标值。可以为负值。

length：用长度值指定坐标值。可以为负值。

left,center right是水平方向取值，而top center bottom是垂直方向的取值。

CSS 复制代码
 
.demo_box{
    position:relative;

    margin:100px auto;
    -webkit-animation:demo_box 4s infinite linear;
    -moz-animation:demo_box 4s infinite linear;
}
.demo_box:hover{
    -webkit-animation-play-state:paused;
    -moz-animation-play-state:paused;
}
.mod_run_code{
    -webkit-perspective-origin:right top;
    -moz-perspective-origin:right top;
    -moz-transform-style:preserve-3d;
    -webkit-transform-style:preserve-3d;
    -webkit-transform: perspective(500px);
    -moz-transform: perspective(500px);
}

@-webkit-keyframes demo_box{
    0%{-webkit-transform:rotateX(0deg);}
    100%{-webkit-transform:rotateX(360deg);} 
}
@-moz-keyframes demo_box{
    0%{-moz-transform:rotateX(0deg);}
    100%{-moz-transform:rotateX(360deg);} 
}
