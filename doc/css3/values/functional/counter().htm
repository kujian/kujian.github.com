counter()

  • 版本：CSS2/3

说明：

用于动态计算长度值。
  • 任何长度值都可以使用calc()函数进行计算。
  • calc()函数支持+, -, *, /, mod运算，以后还可能加入对min()/max()等运算。

兼容性：

插入计数器。
  • 在CSS2.1中counter()只能被使用在content属性上。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
