text-stroke

  • 版本：CSS3
  • 继承性：有

语法：

text-stroke：[ text-stroke-width ] || [ text-stroke-color ]

默认值：看每个独立属性

适用于：所有元素

继承性：有

取值：

[ text-stroke-width ]：
设置或检索对象中的文字的描边厚度
[ text-stroke-color ]：
设置或检索对象中的文字的描边颜色

说明：

复合属性。设置或检索对象中的文字的描边。
  • 对应的脚本特性为textStroke。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6-10 4-18 5.1.7 13 11.5-12.5
