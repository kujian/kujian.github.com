tab-size

  • 版本：CSS3
  • 继承性：有

语法：

tab-size：<integer> | <length>

默认值：8

适用于：块级元素

继承性：有

取值：

<integer>：
用整数值指定制表符的长度。不允许负值。
<length>：
用长度值指定制表符的长度。不允许负值。

说明：

检索或设置对象中的制表符的长度。
  • 该属性决定了制表符(U+0009)的宽度，整数代表空格(U+0020)的倍数(如：tab-size:4; 表示制表符宽度是4个空格的宽度)，不允许负数
  • 对应的脚本特性为tabSize。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6-10 4-18 #1 5.1.7 13-20 10.6-12.5 #1
版本 21-23 #1
  1. 不支持：<length>作为参数值。
