box-orient

  • 版本：CSS3
  • 继承性：无

语法：

box-orient：horizontal | vertical

默认值：horizontal

适用于：弹性盒模型容器

继承性：无

取值：

horizontal：
设置弹性盒模型对象的子元素为水平排列
vertical：
设置弹性盒模型对象的子元素为纵向排列

说明：

设置或检索弹性盒模型对象的子元素的排列方式。
  • 对应的脚本特性为boxOrient。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
较早版本 6-9 4-18 5.1.7 13-23 11.5-12.5
较新版本 10
