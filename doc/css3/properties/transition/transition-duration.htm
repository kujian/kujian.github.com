transition-duration

  • 版本：CSS3
  • 继承性：无

语法：

transition-duration：<time>[ ,<time> ]*

默认值：0

适用于：所有元素，包含伪对象:after和:before

继承性：无

取值：

<time>：
指定对象过渡的持续时间

说明：

检索或设置对象过渡的持续时间。
  • 如果提供多个属性值，以逗号进行分隔。
  • 对应的脚本特性为transitionDuration。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
